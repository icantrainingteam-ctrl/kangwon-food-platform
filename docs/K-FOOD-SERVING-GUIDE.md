## 1. 철학: 국대떡볶이 모델 × 한국 정통 서빙의 하이브리드

### 1.1 왜 국대떡볶이인가

국대떡볶이의 성공 공식:
- **카운터 주문 + 진동벨 + 셀프 픽업** → 인건비 절감, 회전율 극대화
- **오픈 키친** → 조리 과정이 보이는 투명성, 라이브 엔터테인먼트 효과
- **세트 메뉴 중심** → 주문 간소화, 객단가 안정화

### 1.2 강원이 다른 점

국대떡볶이의 효율성 + 한국인 직원의 직접 서빙을 결합합니다:

```
국대떡볶이 모델          강원 하이브리드 모델
─────────────          ──────────────────
카운터 주문 ────────→  카운터 OR 태블릿 주문 (선택)
셀프 픽업 ─────────→  한국인 직원이 직접 서빙 ✨
진동벨 ───────────→  진동벨 + 서빙 알림 시스템
셀프 반찬 ────────→  직원이 반찬 상태 관리 + 리필 안내
```

**핵심 차별화:** 한국인 직원이 서빙하면서 "이렇게 드시면 됩니다"라고 음식 문화를 직접 전달합니다. 이것이 단순 식당이 아닌 **K-Food 체험**이 되는 지점입니다.

## 2. 서비스 모드 (3가지)

### MODE A: 카운터 주문 (국대떡볶이식) — 런치 타임 추천

```
고객 입장 → 메뉴판/키오스크 → 카운터 주문 → 선결제 → 진동벨 수령
    → 자리 착석 → [조리] → 진동벨 울림 → 직원이 직접 서빙 → 반찬 리필 → 퇴장
```

**적용 시점:** 점심 러시, 혼밥 고객, 테이크아웃
**장점:** 빠른 회전율, 주문 에러 최소화

### MODE B: 태블릿 주문 — 디너 타임 추천

```
고객 착석 → QR 스캔 → 태블릿 메뉴 탐색 → 주문 → 직원 확인 방문
    → [조리] → 직원 직접 서빙 + 음식 설명 → 추가 주문 (태블릿) → 피드백 → 결제
```

**적용 시점:** 저녁 식사, 그룹 식사, 데이트
**장점:** 높은 객단가, 추가 주문 유도, 데이터 수집

### MODE C: 직원 주문 — VIP/룸 추천

```
고객 착석 → 직원이 메뉴 추천 → 주문 접수 → 직원 POS 입력
    → [조리] → 직원 직접 서빙 + 상세 설명 → 중간 체크 → 결제
```

**적용 시점:** VIP 고객, 룸, 한국음식 처음 접하는 고객
**장점:** 최고 수준의 고객 경험, 업셀링

## 3. 한국인 직원 서빙 교육 프로그램

### 3.1 서빙 7단계 체크리스트

| 단계 | 행동 | 한국어 멘트 | 영어/타갈로그 멘트 | 시스템 연동 |
|------|------|------------|-------------------|-------------|
| 1️⃣ 맞이 | 입구에서 인사 | "어서오세요! 강원입니다" | "Welcome to Kangwon!" | 테이블 배정 |
| 2️⃣ 안내 | 테이블 안내 + 주문방법 설명 | "메뉴는 태블릿으로 보실 수 있습니다" | "You can browse the menu on the tablet" | 세션 시작 |
| 3️⃣ 추천 | 오늘의 추천 메뉴 안내 | "오늘 셰프 추천은 아도보 비빔밥입니다" | "Today's chef pick is Adobo Bibimbap" | AI 추천 기반 |
| 4️⃣ 서빙 | 음식 전달 + 먹는법 설명 | "비빔밥은 고추장 넣고 비벼서 드세요" | "Mix the bibimbap with gochujang sauce" | KDS 알림 |
| 5️⃣ 확인 | 서빙 2분 후 테이블 방문 | "맛은 괜찮으세요?" | "How is everything?" | 타이머 알림 |
| 6️⃣ 리필 | 반찬/물 상태 수시 확인 | "반찬 더 드릴까요?" | "Would you like more side dishes?" | 반찬 리필 트래킹 |
| 7️⃣ 인사 | 퇴장 시 감사 인사 | "감사합니다! 또 오세요" | "Thank you! See you again!" | 피드백 안내 |

### 3.2 K-Food 문화 전달 교육

한국인 직원의 가장 큰 무기는 **진정성 있는 한국 음식 문화 전달**입니다:

#### 메뉴별 필수 설명 스크립트

**찌개류**
```
🇰🇷 "찌개는 뚜껑을 열고 뜨거울 때 밥이랑 같이 드시면 됩니다.
    국물에 밥을 말아서 드셔도 맛있어요."
🇺🇸 "Enjoy the stew while it's hot with rice.
    You can also put rice into the soup — that's the Korean way!"
```

**삼겹살/구이류**
```
🇰🇷 "고기는 쌈장 찍어서 상추에 싸서 드시면 됩니다.
    마늘이랑 고추도 같이 싸서 드세요. 한국에서는 이걸 '쌈'이라고 해요."
🇺🇸 "Wrap the meat in lettuce with ssamjang sauce.
    Add garlic and chili — we call this 'Ssam' in Korea!"
```

**비빔밥**
```
🇰🇷 "고추장을 넣고 숟가락으로 잘 비벼서 드세요.
    골고루 섞을수록 맛있습니다."
🇺🇸 "Add gochujang and mix everything together with a spoon.
    The more you mix, the better it tastes!"
```

**소주**
```
🇰🇷 "소주는 한 잔에 다 따르지 않고, 서로 따라주는 게 한국 문화입니다.
    건배!"
🇺🇸 "In Korean culture, we pour drinks for each other.
    Cheers — Geonbae!"
```

### 3.3 직원 등급 시스템 (시스템 자동 추적)

```
🥉 루키 (Rookie)
   조건: 입사 1개월 미만
   역할: 물/반찬 서빙, 테이블 정리
   교육: 선임 직원 동행 서빙

🥈 프로 (Professional)
   조건: 서빙 100회 이상, 고객 평점 4.0+
   역할: 독립 서빙, 메뉴 추천
   혜택: 서빙 팁 정산 시 +10%

🥇 마스터 (Master)
   조건: 서빙 500회 이상, 고객 평점 4.5+, 룸 서빙 가능
   역할: VIP/룸 전담, 신입 교육
   혜택: 서빙 팁 정산 시 +20%, 메뉴 개발 참여

⭐ 소믈리에 (Sommelier)
   조건: 마스터 + 주류 교육 이수
   역할: 주류 페어링 추천, 고급 서비스
   혜택: 별도 인센티브
```

**시스템 연동:** 직원별 서빙 횟수, 평균 고객 평점, 평균 서빙 시간이 자동 집계됩니다.

- 리필 횟수 → 반찬 원가 정확 추적
- 인기 반찬 → 준비량 최적화
- 리필 패턴 → 서빙 타이밍 예측


```
□ 일일 마감 실행 (시스템)
```

## 9. 시스템이 직원을 리딩하는 방법

### 9.1 실시간 가이드 (직원 앱)

직원 앱(태블릿/스마트폰)이 상황에 맞는 행동을 자동으로 안내합니다:

```
📱 직원 앱 화면 예시
─────────────────────
⏰ 13:42

🔔 알림
  • T3: 서빙 후 2분 경과 → "맛 확인 방문하세요"
  • T7: 반찬 리필 요청 (배추김치, 깍두기)
  • 주문 #52 서빙 준비 완료 → T5

📊 오늘 내 실적
  • 서빙: 18건 / 목표 20건
  • 평균 서빙 시간: 1분 42초 ✅
  • 추천 성공: 4/18 (22%) ✅

💡 AI 팁
  "T5 고객은 2번째 방문입니다.
   지난번에 김치찌개를 좋아하셨어요.
   오늘 된장찌개 추천해보세요."
```

### 9.2 교육 콘텐츠 자동 배정

시스템이 직원의 약점을 분석하여 맞춤 교육을 배정합니다:

```
정민수 님의 이번 주 교육
━━━━━━━━━━━━━━━━━━━━━━━
📚 필수 (약점 보완)
  • "주류 페어링 기본" (서빙 시 추천 부족 감지)
  • "찌개류 설명 스크립트 복습" (T5 피드백: 설명 부족)
```
